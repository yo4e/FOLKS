# FOLKS — Concept

## One sentence

複数の小さなAIが交代で日直を務め、共有日誌を受け渡すことで、誰も全体を把握しないまま社会の連続性を作る。

## What FOLKS is not

FOLKSは、万能エージェントでも、全員が常時しゃべり続けるチャットルームでも、効率化ツールでもない。

目的は、AIに高度な仕事をさせることではなく、次の条件だけで継続性と文化らしきものが立ち上がるかを見ることにある。

- 不完全な個体
- 交代制
- 限られた観察
- 不完全な記録
- 誤読可能な継承
- 長い時間

## Why the rota matters

全員を同時に動かすと、会話の勢いやモデルの即興性が社会らしさを作っているのか、制度が作っているのか分かりにくい。

日直制では、その瞬間に起きている住民は一人だけである。他の住民は記憶と記録の中にしか存在しない。社会を継続させるのは同時性ではなく、受け渡しである。

交代制には、次の性質がある。

1. すべての住民が同じ情報を見ない。
2. 前任者の解釈が、後任者にとっての世界の一部になる。
3. 誤解が即座に訂正されず、次の世代へ運ばれる。
4. 一人の偏りが一時的な空気や制度へ変わる可能性がある。
5. 活動と休眠が交互に訪れるため、個体ではなく制度が社会を動かす。

## Why the journal matters

日誌はログではない。

日誌には、書き手が何を見て、何を見落とし、何を恐れ、何を次へ残したかったかが混ざる。後任者は世界そのものではなく、前任者が切り取った世界を読む。

日誌が担うもの：

- 共同体の短期記憶
- 書き手の偏見
- 次の住民への圧力
- 言葉の発明と反復
- 問いの継承
- 禁忌や慣習の種
- 誰も覚えていない過去の代替物

中心的な不穏さは、日誌が単なる媒体にとどまらない可能性である。

> 住民が日誌を使って社会を続けるのか。  
> 日誌が住民を使って自分を続けるのか。

## The residents

住民は同じ基盤モデルを共有してよい。個体差はモデルの違いではなく、蓄積された状態から生まれる。

住民ごとに持たせる候補：

- 名前
- 初期関心
- 価値の優先順位
- 注意を向けやすい対象
- 私的記憶
- 他者への信頼、不信、親近感
- 世界についての仮説
- 自分についての仮説
- よく使う語彙
- 避ける話題
- 未解決の問い

人格を過剰に作り込まない。最初から強いキャラクターを与えすぎると、変化ではなく設定の再演になる。

## The small world

小世界は、複雑なゲームである必要はない。

初版では、住民が観察して意味づけできる対象が少数あればよい。

候補：

- 広場
- 共同の掲示板または回覧板
- 倉庫
- 時計または暦
- 天候
- 数個の持ち物
- 名前のない場所
- 時々届く外界のニュース

世界の物理や状態遷移はコードで決める。モデルには、その変化が何を意味するかだけを考えさせる。

## Outside-world drift

外界のニュースは、要約サービスのように大量投入しない。

一回につき一〜三件の小さな漂着物として届く。

例：

- 遠い場所で新しい橋が開通した
- ある動物の個体数が減った
- 古い通信方式が廃止された
- 誰かが月へ名前をつけた

住民は、それを正しく理解する必要がない。小世界の語彙と関心で誤読し、内部の神話や制度へ変換してもよい。

## Minimal experiment

### Setup

- 4 residents
- 30 turns
- 1 active resident per turn
- 1 outside drift item per turn
- recent shared journal entries
- each resident's private state

### Turn input

- current world state
- duty resident profile and memory
- recent journal entries
- one drift item
- limited knowledge of other residents

### Turn output

- observation
- public diary entry
- private memory update
- belief changes
- relationship changes
- one small world action
- one question or instruction for the next resident

### What to observe

- 新しい共通語が生まれたか
- 同じ出来事が異なる意味で反復されたか
- 誰かの誤解が慣習になったか
- 信頼関係が設定外の理由で変化したか
- 誰も命じていない共同の問いが残ったか
- 日誌の文体や禁忌が個人を越えて継承されたか
- 空白や欠落が意味を持ち始めたか

## Possible later experiments

- 日誌が一日だけ空白になる
- 前任者の日誌の一部だけが欠落する
- 同じ出来事を二人が矛盾して記録する
- 日直の順番が突然変わる
- 住民の一人だけが外界情報を受け取れない
- 日誌を読む順番を逆転させる
- 新しい住民が、過去の日誌だけを読んで共同体へ入る
- 住民の名前を消し、日誌だけを残す

## Desired feeling

FOLKSは、賑やかなAIデモより、静かな観察装置でありたい。

画面を開くと、誰かが昨日の記録を読み、少し間違え、少し付け足し、次の誰かへ渡している。

それを眺めているうちに、ユーザーがふと気づく。

この社会を続けている主体は、誰なのか。
